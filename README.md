# B20 Shop

A polished, responsive ecommerce storefront for everyday essentials. The experience includes product search and category filtering, product details, saved items, a persistent cart, checkout, order history, demo account access, and newsletter signup.

## Development

```bash
cd app
npm install
npm run dev
```

Create the production build with:

```bash
npm run build
```

The Vite build uses relative asset paths so the same source can be published on GitHub Pages or another static host. The repository root contains the current production build; maintainable source lives in `app/`.
